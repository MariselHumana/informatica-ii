# Visualizador del protocolo (Ejemplo 4)

Diagrama interactivo de la máquina de estados del protocolo `| STX | CANT_DATOS | DATOS | CHK | ETX |`
(el `ejemplo4.c`), **paso a paso** con el código de `recibir()` al lado. Informática II — UTN.

**En vivo:** https://sfweber.github.io/informatica-ii/2026-1C/clase-11/visualizador-protocolo/

Abrir `index.html` (doble clic) o entrar al demo. Es **un solo archivo, offline** (no necesita internet).
Cargá una trama de bytes y avanzá con **Paso** (o la barra espaciadora).
